# CSV Plotting App

## Description

This project implements an **interactive CSV visualization tool** built with **Dash** and **Plotly**.  
The application allows users to **upload CSV files**, select which columns to plot, and visualize them dynamically through **interactive line charts**.  
Multiple files can be uploaded, stored, and explored within the same session.

## Installation

To run the code, install the following Python packages:  
pip install dash plotly pandas

## Usage

1.  Run the script:  
    python csv_plotting_app.py
    
2.  Open your browser and navigate to:  
    [http://127.0.0.1:8050/](http://127.0.0.1:8050/)
    
3.  Drag and drop a `.csv` file or click “Select CSV Files” to upload.
    
4.  Choose the X-axis column and one or more Y-axes to visualize your data interactively.
    

## Features

-   Upload and parse CSV files directly in the browser.
    
-   Store and manage multiple datasets simultaneously.
    
-   Dynamic axis selection for flexible plotting.
    
-   Interactive, dark-themed Plotly visualizations.
    
-   Clean and minimal Dash interface for quick data exploration.
